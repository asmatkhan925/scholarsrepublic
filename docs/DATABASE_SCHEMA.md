# Database Schema

Database: PostgreSQL

Phase 1 creates the Django project and app boundaries. MVP models will be implemented phase by phase.

Planned main models:

- `users.User`
- `profiles.StudentProfile`
- `scholarships.Scholarship`
- `scholarships.SavedScholarship`
- `applications.Application`
- `services.ServiceRequest`
- `blog.BlogPost`

Main models should include `created_at` and `updated_at` timestamps where applicable.
